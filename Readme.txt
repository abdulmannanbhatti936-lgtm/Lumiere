# Lumiere Stays - 3D Luxury Hotel Booking Platform

A premium, immersive hotel booking platform featuring interactive 3D experiences, built with React, Three.js, and modern web technologies.

## 🌟 Features

- **Immersive 3D Experiences**
  - Interactive 3D hotel models with real-time rendering
  - 3D room previews with detailed visualization
  - Interactive 3D globe for destination browsing
  - Smooth parallax scrolling effects

- **Luxury Hotel Browsing**
  - Advanced search and filtering by location, price, and ratings
  - Detailed hotel information with image galleries
  - Amenities showcase with icons
  - Star rating system

- **Seamless Booking Flow**
  - Interactive date picker for check-in/check-out
  - Guest counter with capacity limits
  - Multi-step booking confirmation
  - Booking summary with price breakdown

- **User Reviews & Ratings**
  - Submit and view guest reviews
  - Star rating system (1-5 stars)
  - Review aggregation and display

- **Admin Dashboard**
  - Analytics with charts and graphs
  - Booking management
  - Revenue tracking
  - Real-time statistics

- **Responsive Design**
  - Mobile-first approach
  - Tablet and desktop optimization
  - Touch-friendly interfaces

- **Dark Luxury Theme**
  - Premium color palette
  - Smooth animations and transitions
  - Elegant typography
  - Refined UI components

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

1. Clone the repository:
\`\`\`bash
git clone https://github.com/yourusername/lumiere-stays.git
cd lumiere-stays
\`\`\`

2. Install dependencies:
\`\`\`bash
pnpm install
\`\`\`

3. Start the development server:
\`\`\`bash
pnpm dev
\`\`\`

4. Open your browser and navigate to \`http://localhost:5173\`
